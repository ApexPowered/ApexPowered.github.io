ApexOS beta builds now available.

Feb 5th, 2016

ApexOS beta builds are now available from the installer by running the pastebin command like this:

pastebin run XXXXXX -s unstable

This requires PatchUtils to be installed, if that is not found, it will be downloaded during installation.

Installation takes nearly seconds and updates are rolling release, you can revert
back to older builds or change your build stream (release or unstable) in the Apex
Updater. You can get applications from the Apex Appstore in your applications menu.
New builds will be released over time and ApexOS will check for updates every boot,
if your on unstable, please report bugs to our issue tracker

And for ApexPKG, our awesome package manager, report bugs here

Anyway, test the build if you want, please report the bugs if you can. This project
is still in development, so give me some slack. If you want to contribute, PM me
on the forums. My username is TheSyndicate1.
